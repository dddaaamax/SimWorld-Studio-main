# SimWorld Studio — Web Stack

Full-stack web application: React frontend + Node.js/Express backend.

## Structure

```
web/
├── server/          Backend (Node.js + Express)
├── src/             Frontend (React)
├── public/          Static assets (ue-player.html, ue-assets/player.js)
├── dist/            Built frontend (auto-generated by `npm run build`)
├── vite.config.js   Build config
└── package.json     Frontend dependencies (React, Vite, ReactMarkdown)
```

## Development

```bash
# Build frontend
cd web
npm install
npm run build

# Start server only (frontend pre-built)
cd server
node index.js

# Hot-reload frontend (dev mode)
npm run dev     # Vite dev server on :5173 with proxy to :3002
```

## Ports

| Service | Default Port | Description |
|---|---|---|
| Web Server | 3002 | Express API + static frontend |
| Cirrus HTTP | 8685 | Pixel Streaming player endpoint |
| Cirrus WS | 8686 | Pixel Streaming streamer WebSocket |
| Cirrus SFU | 8989 | Selective Forwarding Unit |

| Variable | Default | Description |

| `PORT` | 3002 | Web server port |

| `UNREAL_PORT` | 55557 | UE MCP TCP port |
| `UCV_PORT` | 9001 | UnrealCV port |
| `CIRRUS_HTTP_PORT` | 8685 | Cirrus HTTP port |
| `CIRRUS_WS_PORT` | 8686 | Cirrus WS port |
| `PIXEL_STREAMING_URL` | http://127.0.0.1:8685 | Pixel streaming base URL |
| `NODE_ENV` | production | production / development |
| `CLAUDE_BIN` | claude | Claude Code CLI binary path |
| `SIMWORLD_ASSETS_FILE` | assets_full.json | Asset catalog file |
